数字型

type desc
(u)int (无符号)不规定长度，在32位系统就32位，64位系统就64位
(u)int8 (无符号)规定长度8
(u)int16 (无符号)规定长度16
(u)int32 (无符号)规定长度32
(u)int64 (无符号)规定长度64
unitptr 指针，长度跟操作系统相关

字符型

type desc
byte 长度8位（8bit）
rune 长度32位
ps: rune 相当于 go 的 char

浮点型

type desc
float32
float64
complex64 64位的复数
complex128 128位的复数

switch 的 fallthrough 语句

Go里面switch默认相当于每个case最后带有break，匹配成功后不会自动向下执行其他case，而是跳出整个switch, 但是可以使用fallthrough强制执行后面的case代码
