### 1、下载地址
[https://studygolang.com/dl](https://studygolang.com/dl)

### 2、选择合适平台后安装，后验证是否安装完成
```shell
go version
```

### 3、设置 go module 为开启状态
```shell
go env -w GO111MODULE=on
```

### 4、设置 go package 国内镜像地址
```shell
go env -w GOPROXY=https://goproxy.cn,direct
```

### 5、Done
