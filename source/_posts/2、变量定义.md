## 例子
```go
// 正常定义（自动类型推断）
var aa = 1
var bb = "kan"

// 简略定义（自动类型推断）
var (
    cc = 2
    dd = 3
)

// 完整定义
var a bool = true
var b string = "ha"
var c int = 123

// 缩写定义（只能函数内）
g := true
h := "ha"
i := 123

// 联合定义
var j,k,l = true,123,"xi"
```

## 打印相关 fmt.Printf
| code   | desc         |
| ------ | ------------ |
| %v     | 原样输出     |
| %T     | 打印类型     |
| %t     | bool类型     |
| %s     | 字符串       |
| %f     | 浮点         |
| %d     | 10进制的整数 |
| %b     | 2进制的整数  |
| %o     | 8进制        |
| %x、%X | 16进制       |
| %x     | 0-9，a-f     |
| %X     | 0-9，A-F     |
| %c     | 打印字符     |
| %p     | 打印地址     |
```
