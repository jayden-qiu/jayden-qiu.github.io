### 3、数字型
| type     | desc                                                 |
| -------- | ---------------------------------------------------- |
| (u)int   | (无符号)不规定长度，在32位系统就32位，64位系统就64位 |
| (u)int8  | (无符号)规定长度8                                    |
| (u)int16 | (无符号)规定长度16                                   |
| (u)int32 | (无符号)规定长度32                                   |
| (u)int64 | (无符号)规定长度64                                   |
| unitptr  | 指针，长度跟操作系统相关                             |

### 4、字符型
| type | desc            |
| ---- | --------------- |
| byte | 长度8位（8bit） |
| rune | 长度32位        |
*ps: rune 相当于 go 的 char*
